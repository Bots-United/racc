    PLEASE NOTE
         The RACC project is still in its early stages of development. It is NOT a final version. This means that many features are still left to be implemented and that you must not not expect something perfect (yes, it seems obvious as it is, but seeing some e-mails people send me, sometimes it makes me wonder !). Notably the following features are not correctly implemented yet :
         - Learning
         - Memory
         - Emotions
         - Sense of orientation
         - Sense of duty
         - Auto teamup

         THIS MEANS that the fact you don't see those features IS NOT A BUG. If you don't see the bots rescue the hostages in Counter-Strike, this is normal. If you don't see the bots capture the flag in Team Fortress Classic, this is normal. Let me with enough time to program them, or do it yourself. The source code is available.

         Secondly, I want you to keep in mind that comparing these bots with the other waypointed bots is pointless. Keep in mind that it is a totally different navigation, which attempts to use human techniques.

    The bots sample their field of view in real-time

         THIS MEANS that their navigation is less good than those of the POD-Bots with whom everybody seems to be willing to compare them with. If you don't see the bots climb the roofs in Counter-Strike, this is normal. If you don't see the bots walk through some difficult areas, this is normal. The power of nowaday's computers does not just allow their artificial intelligence, copied from the human model, to behave as good as those of the waypointed bots, which bots don't even need eyes to follow a path.

         You have been warned.
