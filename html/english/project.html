• The RACC project

    CURRENT RELEASE: 20020509
    Check out the changelog

         The Rational Autonomous Cybernetic Commandos (RACC) is a bot AI development project which aims at implementing as close as possible a human-like behaviour for NPCs (non-player characters) in first-person shooter games such as Half-Life. The current release of the project supports the following mods: Half-Life, Team Fortress Classic, Opposing Force, Azure Sheep, Deathmatch Classic and Counter-Strike (soon War In Europe). The code layout is based on the HPB template, a work done by Botman permitting an efficient interfacing between the Half-Life game engine and almost any MOD DLL. Botman kindly provided his code to the Half-Life coding community, and many developers used it to create their own plugin. Bots are an example.
         The RACC project has been wrote under the BSD License, to allow anyone to use the source code and do whatever he wants with it, provided that the original source of the code is explicitly mentioned in it, i.e. Botman's HPB.

  • Features

    The RATIONAL AUTONOMOUS CYBERNETIC COMMANDOS bot AI features are the following:

      Bot setup and monitoring:
    • Support for Half-Life, Opposing Force, Team Fortress, Counter-Strike, Deathmatch Classic and Azure Sheep
    • Clean installation centralized in a single directory not to corrupt any Half-Life file
    • Easy setup and removal with two batch scripts: 'install.bat' and 'remove.bat'
    • High-quality additional player models are included with the distribution
    • Automatic in-game bot spawning and removal equalized to max player count
    • Automatic bot team assignment for team-based MODs
    • Console tools to monitor and control bot characteristics and bot counts
    • Bot personalization system involving bot name, skin and skill

      Chat:
    • Advanced bot text chat system with up to 12 text chat situations
    • Bots talk like if they were using Half-Life's voice chat system in team-based MODs
    • Bots use the radio in Counter-Strike
    • Bots can ask for backup when they get damaged and call for medic when permitted

      Combat behaviour:
    • Advanced weapon selection and usage, including burst-firing and grenade handling
    • Human-like aiming system allowing to aim at most of the parts of a body
    • Unpredictable combat behaviour, involving fearful and rusher bots
    • Squad intelligence for bots to decide to set up as a squad and cover their backs
    • Advanced bot buy routines in Counter-Strike
    • Bomb planting and defusal implemented as map goal in Counter-Strike
    • Bots may call each other when they see the VIP or the bomb site in Counter-Strike
    • Advanced sniper rifle and crossbow usage, and grenade/satchel/snark avoidance
    • Bots spray logos (can't change decal yet)

      Navigation:
    • TOTALLY UNWAYPOINTED NAVIGATION: bots sample their field of view in real-time like humans do
    • Careful look for potentially dangerous corners while wandering; bots can strafe or turn there
    • Basic sense of orientation implemented for bots to remember interesting locations
    • Bots strafe to avoid teammates while wandering
    • Bots are able to see up a slope or a staircase
    • Bots can open doors, break windows and walk through them
    • Bots avoid falling more cleverly

      Player/Bot interaction:
    • VGUI menu based interface and keybindings to pass orders to bots from your team
    • Orders understood by a bot: 'Follow me', 'Stop following me', 'Keep the position'
    • Advanced follow-user code: on ladders, over crates and through windows if their user jumps or ducks
    • Used bots pausing will try to cover their backs and listen to squad members behaviour

      Miscellaneous:
    • Strong emphasis on source code correctness, cleanliness and commenting


    Counter-Strike: a bot keeping the place


  • What's Half-Life ? What's Counter-Strike ?

         Half-Life is a first-person shooter video game, meaning that the player is viewing the game world by his own character's eyes, this producing an important feeling of realism. The goal of the game for the player is to make use of weapons he collects in the game world to destroy his enemies, which can be computer-controlled monsters (in single player mode) or real player opponents (in multiplayer mode). People can play over the Internet, or in LAN-parties (small temporary dedicated local networks), and so kick friendly ass all night long.

         Counter-Strike is the most famous MOD ("modification" of the game) for Half-Life in which people play with new player models, new guns, new environments, but mostly in a new context. Conter-Strike is a team-based game where terrorists and counter-terrorist teams fight each other. That MOD makes a very great deal of realism, thus headshots and falls from high places are deadly.

  • What's a bot ?

         Bot means robot. A bot is a computer-controlled opponent, with whom people who aren't connected to the Internet or those who just missed the latest LAN-party can train a little, while waiting for the next one. A bot has the same look as a human player, and its AI program is aimed at simulating a normal player's in-game behaviour : fire at you when he sees you, climb at ladders, hide under a staircase while waiting for you, grab items, laugh at you when you are dead, in short all possible & imaginable treacheries like in a true network game. If the bot is good, the player could have the feeling that he is playing against a human opponent.

         A bot's AI is way more complex than the one of the game's standard monsters, which can seem fairly stupid in general. The goal for the developer consists in making his bot think in real time : make him able to identify a wall, a door, his weapon, an enemy firing at him, make him know that if he falls from high he'll probably get some damage, make him able to either go and hide or attack when taken under heavy fire, know which is the best weapon to use in whatever situation, know when to burst-fire, aim accurately or throw a grenade, and a bunch of obvious behaviours that have to be correctly implemented in the AI in an attempt to make his bot if not clever, at least less stupid, again and again.
